## Project Description
MediCore is a console-based Hospital Management System built in C++.  
It supports three user roles: **Patient**, **Doctor**, and **Admin**.  
All data is stored in and read from `.txt` files so it persists across restarts.


## How to Compile and Run

1. Open `OOP FINAL PROJECT 2026.sln` in Visual Studio 2022
2. Set configuration to **Debug** and platform to **x64**
3. Make sure all `.txt` files are placed in the **project folder** (same level as `.vcxproj`)
4. Press **Ctrl + Shift + B** to buildS
5. Press **Ctrl + F5** to run
